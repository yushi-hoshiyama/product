## 1. コンセプト

### サービス定義
DELNEは、認知症のご家族からの**頻回電話**を**家族の声で再現したAI**が24時間365日代わりに受電するサービス。ご家族の電話対応負担を軽減し、仕事と介護の両立を支援する。
第1期で「家族の声AIが認知症患者に受け入れられる」ことを実証済み。第2期では「ケアの質の継続的向上」と「組織運用の実現性」を検証する。

### キャッチコピー・主要メッセージ（LP）
- **メインコピー**：「でんわにデルネ」
- **サブコピー**：「介護と仕事の両立を支援するAI電話サービス」
- **約束**：「認知症のご家族からの電話にAIが24時間365日丁寧に応答」


### ターゲットユーザー（3ペルソナ・LP定義）

| ペルソナ | 状況 | 提供価値 |
|---|---|---|
| **① 直接介護されている方** | 鳴りやまない親からの電話で優しく話せず自分を責めている | 心と時間を守る。仕事中・睡眠中の対応負担を解消 |
| **② 経営者・人事担当者** | 優秀な社員が親の介護で仕事を中断・退職している | 福利厚生としての導入で介護離職防止 |
| **③ 遠方在住で直接介護が難しい方** | 地元で介護する兄弟姉妹に申し訳なさを感じている | 費用負担という形で介護を支援（→第2期プレゼント機能で強化） |


### 解決する課題
認知症介護の構造的課題、認知症者による頻回電話を技術と経済の両側面から解消する。

- **当事者**：認知症患者本人 + 5つの当事者（介護家族／ケアマネ／企業／地域／行政）
- **負担集中**：仕事と介護を両立する約307万人のビジネスケアラーに負担が集中（年間10万人が介護離職、生産性損失9兆円／2030年推計）
- **広島固有性**：広島県は認知症患者率全国ワースト1位（65歳以上女性6.39%）、認知症患者約15万人 + MCI約13万人

### コアバリュー
「ケアの孤立」を技術的・経済的の両側面から解消し、社会インフラとして再設計する。

### 競合優位性（カテゴリ比較）
| 軸 | DELNE+ |
|---|---|
| 音声 | 家族の声を再現（汎用AI音声ではない） |
| 会話設計 | バリデーション技法 + 回想法（医療/学術知見ベース） |
| 社会実装 | 医療法人で認知症患者に実証済み（健常者向け汎用ではない） |
| 支払構造 | プレゼント機能（個人課金のみではない） |
| 想定市場 | 認知症予備軍・認知症患者特化 |

## 2. 機能

### 現行運用機能（LP掲載・5機能）

| 機能 | 内容 |
|---|---|
| **簡単操作** | 電話番号そのままで電話するだけで利用可能 |
| **「あなた」の声で応答** | 介護家族の声をAIが学習し、自然な会話を実現 |
| **24時間365日対応** | どんな時でも自動応答 |
| **感情ケア** | 医療機関監修の専門知見（バリデーション技法）でやさしく応答 |
| **会話内容を通知** | 着信通知だけでなく、会話の要約をSMS/メールで送信 |

**運用上の工夫**：
- 事前にご家族の予定（デイサービス等）を登録すると、AIが状況に合わせて回答
- 認知症の方に合わせた「口調」と「速度」で応答（健常者には少し緩やかに感じる程度）

### 利用開始フロー（3ステップ・LP）

```
STEP 1  アカウント登録（無料トライアルから5分で登録）
   ↓
STEP 2  音声登録（録音した声をAIが学習し、再現）
   ↓
STEP 3  利用開始（DELNEが発行する「専用番号」にかけるだけ）
```

### デモ・体験チャネル（LP）
- **デモ専用ダイヤル**：050-3515-7477（24時間対応／通話料お客様負担／標準語応答）
- **サンプル音声試聴**：LP上で再現音声サンプルを聴取可能
- **無料トライアル**：1ヶ月無料（2026年5月申込限定キャンペーン延長中）


### 第2期新機能

#### ① 認知症進行抑制機能を持つ音声会話AI
- 介護家族・大学・医療機関と連携し、**回想法等の非薬物療法**をAIに学習
- **2つのAIを並列運用**：
  - 対話用リアルタイム会話AI（既存）
  - スコアリング用高精度AI（新規：会話分析・ライフログ蓄積・話題誘導指示）
- **AI側からの架電**：服薬確認・起床確認等の生活改善・見守り

#### ② ライフログ基盤
通話の副産物として3層のデータが自然に蓄積：
1. 通話メタデータ（日時・話題）
2. 会話パターン（発話量・感情トーン）
3. 生活リズム（睡眠・食事 ※会話から間接取得）

4ステークホルダーへの価値提供：
- 患者：会話の毎日最適化・スコアリング
- 家族：ナラティブ形式のレポート、複数指標逸脱時の緊急通知
- ケアマネ：ケアプランへの活用、申し送り資料化
- 社会：匿名化ビッグデータで産学連携活用

#### ③ プレゼント機能
「誰が使うか」と「誰が払うか」を分離。第三者がDELNE+を介護家族にプレゼント可能。

| STEP | 支払者 | 形態 |
|---|---|---|
| 1 | 介護家族本人 | 月額サブスク |
| 2 | 親族・知人 | プレゼントコード前払い |
| 3 | 企業 | EAP / カフェテリアプラン |
| 4 | 行政 | 助成金・補助金（両立支援等助成金最大¥60万/社） |

充当順序：①ギフト残高 → ②企業スポンサー → ③主契約者自己負担

## 3. 開発方針

### 開発スタイル
**少数精鋭 × アジャイル × AI駆動開発**
- 各フェーズでアジャイル開発、機能フェーズ毎に実証実験
- 開発・営業・保守をAIエージェント（Claude Code等）で半自動化
- 組織運営のAX（AIトランスフォーメーション）自体も実証対象

### 体制
- **AI技術者内製**
- 株式会社オノゴロ：代表 + PM/AIエンジニア + マーケ/AIエンジニア + 営業/AIエンジニア（広島本社、広島県内採用計画）
- 松栄会：認知症介護アドバイザー（実証フィールド提供）

### 開発フェーズ（2026年〜2027年）
| フェーズ | 内容 |
|---|---|
| Phase 1 | 認知症抑制機能開発 |
| Phase 2 | ライフログ開発 |
| Phase 3 | アプリ開発 |
| Phase 4 | プレゼント機能開発 |
| Phase 5 | 架電AIシステム開発 |

各フェーズに対応する4つの実証実験（A: 認知症抑制、B: ライフログ、C: プレゼント+アプリ、D: 架電AI）。

### 技術選定の根拠
- 通話料・AI利用料を実測ベースで損益分岐計算済み
- 変動費単価はNTT CPaaS・Google AI・Fish Audio等の公開料金表に基づく実コスト
- 架電通話料は受電の3.7倍（¥17.82/分 vs ¥4.82/分）→ 1回3〜5分自動制限でコスト予測可能化

### 連携パートナー
- NTTドコモビジネスX（電話インフラ）
- 株式会社AIデータ（AIデータ基盤）
- 松栄会（介護）
- 介サポひろしま（介護施設連携ハブ）
- 日本福祉用具供給協会（福祉製品販売ハブ）

### 4チャネル販売戦略（月次獲得目標：54件/月）
1. **BtoCデジタルマーケティング**（15件 + 紹介10件/月）：LP・SNS広告・SEO
2. **地域包括/認知症カフェ直接営業**（20件 + カフェ4件/月）：自治体一括導入（10〜50世帯単位）も視野
3. **企業スポンサー開拓**（5件/月）：人事・福利厚生部門へのダイレクトマーケ
4. **プレゼント機能による紹介**：利用者・企業スポンサーからの横繋がりNW（①〜③が育つほど加速）

### 広島の地域パートナー戦略
- 介護家族（ビジネスケアラー）：デジタルマーケ + コミュニティで直接アプローチ
- 介護施設：ケアマネ経由でアプローチ
- 民間企業：補助制度活用で中小企業（広島県内99%）にアピール
- 認知症支援組織/団体：認知症カフェ等を口コミ拡散の起点に
- 地域包括支援センター（広島41か所）：行政との窓口

### 訴求ポイント
- **企業向け**：介護離職1人あたりの企業損失約500万円 vs DELNE導入コストでROI明示
- **法改正タイミング**：2025年4月育児・介護休業法改正で雇用環境整備が義務化
- **初期導入コストゼロ**：1アカウント単位購入可能、中小企業の福利厚生インフラに

3. **中小企業の支援空白**：両立支援「何もしていない」28%、1アカウント単位購入で導入容易化

## 補足：用語

- **バリデーション技法**（Naomi Feil, 1963〜）：認知症患者の感情を否定せず、共感・傾聴で受け止める対話技法
- **回想法**：過去の記憶に触れる会話で脳を活性化する非薬物療法
- **ビジネスケアラー**：仕事を持ちながら家族の介護も行う人
- **MCI**：軽度認知障害（認知症の予備軍）
- **EAP**：従業員支援プログラム
