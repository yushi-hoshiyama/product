# 表現コンプライアンス規程

DELNEのマーケティング・PR表現における表現規範。
全てのコンテンツ (LP・広告・プレスリリース・SNS投稿・営業資料等) 作成時に本規程に照らす。

## 0. 適用範囲

DELNEに関する **全ての対外的表現**。社内資料はこの限りではないが、社外に出る可能性のあるものは原則本規程に従う。

## 1. 基本方針

1. **景品表示法・薬機法・医療広告ガイドライン** の精神を守る
2. **認知症の方の尊厳** と **家族介護者の心理** に配慮する
3. **検証可能な事実** のみを訴求し、未検証の効果は謳わない
4. グレーゾーンに踏み込む前に、必ず本規程と照合する

## 2. DELNE の訴求軸 (重要)

DELNE の主訴求軸は **家族介護者のウェルビーイング向上** である。
認知症進行抑制の技術は内部に持つが、現時点では**訴求しない** (長期目標として温存)。

### ✅ 積極訴求OK (主訴求)

| 訴求テーマ | 表現例 | 根拠 |
|---|---|---|
| 家族介護者の負担軽減 | 「家族の電話対応負担を軽減」 | 機能事実 (頻回電話の代行) |
| 心理的ケア (家族側) | 「介護家族の心の余裕を取り戻す」 | 機能事実 |
| 安否確認・見守り | 「定期的な電話で安否を確認」 | 機能事実 |
| 話し相手・会話 | 「AIが話し相手になる」 | 機能事実 |
| リマインド支援 | 「服薬・通院をお声がけ」 | 機能事実 |
| 状態レポート | 「会話の内容を家族にレポート」 | 機能事実 |
| 介護施設監修 | 「○○介護施設監修」 | 事実 (具体名を明示すること) |

### ⚠️ 慎重な表現 (使い方に注意)

| 表現 | 注意点 | 推奨言い回し |
|---|---|---|
| 「孤独感の軽減」 | 主観的体感の断定はNG | 「孤独感の軽減につながる可能性」 |
| 「ウェルビーイング向上」 | 効果の断定を避ける | 「ウェルビーイングの向上を目指す」 |
| 「認知機能のモニタリング」 | 「測定」「診断」を示唆しない | 「会話パターンの変化を記録」 |
| 「医療機関と連携」 | 連携が**事実化してから**使う | 現時点では「連携を予定」「協議中」程度 |
| 「大学と共同研究」 | 締結後に使う | 現時点では使用禁止 |
| 「介護のプロが監修」 | 監修者の専門性に応じて | 具体名を出す方が安全 |

### ❌ NG表現 (絶対に使わない)

#### 医療・治療関連
- 「認知症の予防」「認知症を予防」
- 「認知症の改善」「認知症が改善」
- 「認知症の治療」「治療効果」「治療メソッド」
- 「進行を遅らせる」「進行抑制」「進行を防ぐ」
- 「認知機能の向上」「認知機能の改善」
- 「医療機器」「医療レベル」「医療的ケア」
- 「医師監修」 (医師による監修が事実でない限り)
- 「○○病院推奨」「○○学会認定」 (事実でない限り)

#### 医行為類似
- 「診断」「判定」「測定」(認知症の)
- 「重症度判定」「進行度測定」
- 「兆候を発見」「異常を検出」(医療的文脈で)
- 「スクリーニング」(医療スクリーニング的文脈で)

> 💡 進行度スコアリング機能は内部品質指標として実装するが、「診断」「測定」と読める表現での訴求はしない。家族向けレポート機能としても「会話の様子をお知らせ」程度にとどめ、「進行度」「重症度」を直接表示しない。

#### 優良誤認・有利誤認 (景表法)
- 「必ず」「絶対」「100%」「確実」 (効果・サービス品質の断定)
- 「日本一」「業界初」「No.1」 (客観的立証なし)
- 「世界唯一」「他にはない」 (立証なし)
- 「最先端」「最新技術」 (具体的根拠なし)
- 「劇的に」「驚くほど」 (体験談を装った誇大表現)

#### 配慮欠如表現
- 「認知症患者」 → 「認知症の方」「認知症のある方」
- 「ボケ」「呆ける」「ボケ防止」 → 使用禁止
- 「徘徊」 → 「ひとり歩き」(可能なら言い換え)
- 「問題行動」 → 「行動・心理症状(BPSD)」または具体的描写

#### 家族介護者への罪悪感喚起
- 「あなたが見ていないと…」
- 「忙しいあなたに代わって…」(罪悪感調)
- 「ご家族の責任で…」
- 「親を一人にしないで…」

> 💡 家族介護者向けマーケティングでは「あなたを責めるトーン」ではなく「あなたを支えるトーン」を選ぶ。

## 3. 必須記載事項

以下を訴求する場合は、必ず根拠/注釈を併記する。

| 訴求 | 必須記載 |
|---|---|
| 「○○名以上のご利用」 | 利用者数の出典・集計時点 |
| 「△△%が満足」 | 調査対象・期間・有効回答数 |
| 「介護施設監修」 | 監修施設の具体名 (許可済の場合) |
| 「効果がある」系の表現 | 根拠 (調査・データ・出典) |
| お客様の声 | 個人特定情報の同意取得済表記 |
| 比較表現 (他社との比較) | 比較対象の明示・調査時点 |

## 4. チェック手順

### 4.1 制作時セルフチェック

コンテンツドラフト完成後、以下を確認:

1. **NG表現リスト** にヒットする語が含まれていないか (検索)
2. **慎重な表現** を使っている場合、推奨言い回しに修正したか
3. **必須記載事項** が漏れていないか
4. 「**家族介護者を責めるトーン**」になっていないか
5. 認知症の方を **当事者として尊重した表現** になっているか

### 4.2 公開前最終チェック

- 法令解釈に不安がある表現は、本規程未収載でも保留にする
- 大規模公開 (プレスリリース・大型広告等) は、社内で複数人レビュー

## 5. 例外承認プロセス

本規程と外れた表現を使いたい場合:

1. なぜその表現を使いたいか・代替案がないかを整理
2. 法的リスク評価 (景表法・薬機法等の観点)
3. 個別判断として承認/却下
4. 承認された場合は本規程の「例外事例」セクションに追記 (将来の参照のため)

## 6. 更新ポリシー

本規程は **固定ではなく、サービス成長・法令変化に応じて更新する**。

### 将来解禁を検討する領域

| 領域 | 解禁条件 |
|---|---|
| 認知機能への効果訴求 | 査読論文・RCT等のエビデンス確立後 |
| 「医療機関連携」 | 連携契約締結後・連携先名公表許可後 |
| 「大学共同研究」 | 共同研究契約締結後 |
| 「医師監修」 | 監修体制の整備後 |
| 進行度スコアリングの可視化 | 医療法務レビュー・適切な責任分界後 |
