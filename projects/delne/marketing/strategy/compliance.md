# 表現コンプライアンス規程

DELNEのマーケティング・PR表現における表現規範。
全てのコンテンツ (LP・広告・プレスリリース・SNS投稿・営業資料等) 作成時に本規程に照らす。

> 📌 製品事実は `projects/delne/product/summary.md` を single source of truth として参照する。
> 本規程は **どう表現するか** のルールであり、**何があるか** は summary.md を参照。

## 0. 適用範囲

DELNEに関する **全ての対外的表現**。社内資料はこの限りではないが、社外に出る可能性のあるものは原則本規程に従う。

## 1. 基本方針

1. **景品表示法・薬機法・医療広告ガイドライン** の精神を守る
2. **認知症の方の尊厳** と **家族介護者の心理** に配慮する
3. **検証可能な事実** のみを訴求し、未検証の効果は謳わない
4. グレーゾーンに踏み込む前に、必ず本規程と照合する

## 2. DELNE の訴求軸

DELNE の主訴求軸は **家族介護者のウェルビーイング向上** (頻回電話負担の軽減・介護離職の防止)。
副次訴求として **認知症の方への会話・見守り価値** がある。

認知症進行抑制の技術 (スコアリングAI・非薬物的アプローチ) は内部に持つが、
**「進行抑制」「進行を遅らせる」等の効果断定表現はしない** (将来エビデンス確立後に解禁検討)。

### ✅ 積極訴求OK (主訴求)

| 訴求テーマ | 表現例 | 根拠 |
|---|---|---|
| 家族介護者の負担軽減 | 「家族の電話対応負担を軽減」「介護と仕事の両立を支援」 | 機能事実 (頻回電話の代行) |
| 心理的ケア (家族側) | 「介護家族の心と時間を守る」「優しく話せず自分を責めるあなたへ」 | 機能事実 |
| 安否確認・見守り | 「定期的な電話で安否を確認」「AI側からの架電で見守り」 | 機能事実 |
| 話し相手・会話 | 「あなたの声でAIが応答」「24時間365日対応」 | 機能事実 |
| リマインド支援 | 「服薬・通院をお声がけ」 | 機能事実 |
| 状態レポート | 「会話の要約をSMS/メールで通知」「会話のようすをナラティブでお知らせ」 | 機能事実 |
| **医療法人監修** | 「医療法人社団 松栄会 監修」「認知症介護パートナー: 医療法人社団 松栄会」 | 事実 (松栄会と提携済) |
| バリデーション技法・回想法 | 「バリデーション技法を活用」「回想法を活用した会話設計」 | 学術的技法名 (効果の断定は別問題) |
| 介護離職防止 | 「介護離職の防止」「優秀な人材の離職を防ぐ福利厚生」 | 機能事実 (企業向け) |
| 公的採択 | 「ひろしまAIサンドボックス採択プロジェクト」 | 事実 |

### ⚠️ 慎重な表現 (使い方に注意)

| 表現 | 注意点 | 推奨言い回し |
|---|---|---|
| 「孤独感の軽減」 | 主観的体感の断定はNG | 「孤独感の軽減につながる可能性」 |
| 「ウェルビーイング向上」 | 効果の断定を避ける | 「ウェルビーイングの向上を目指す」 |
| 「大学共同研究」 | 締結後に使う | 現時点では使用禁止 (連携予定の段階) |
| 「非薬物療法」 | 「療法」は医療類似と取られうる | 「**非薬物的アプローチ**」「**学術的会話設計**」 |
| 「ライフログ」「生活リズム取得」 | プライバシー誤解を生まないよう同意取得を明示 | 「ご家族の同意のもと会話から間接取得」 |
| 「いつもと違う様子の通知」 | 医療的アラートと誤認させない | 「ふだんと違う会話の様子をお知らせ」(医療的判断ではないことを示唆) |

### ❌ NG表現 (絶対に使わない)

#### 医療・治療関連

- 「認知症の予防」「認知症を予防」
- 「認知症の改善」「認知症が改善」
- 「認知症の治療」「治療効果」「治療メソッド」
- **「進行を遅らせる」「進行抑制」「進行を防ぐ」「進行を緩やかに」**
- 「認知機能の向上」「認知機能の改善」
- 「医療機器」「医療レベル」「医療的ケア」
- 「医師監修」 (医師による監修が事実でない限り)
- 「○○病院推奨」「○○学会認定」 (事実でない限り)

#### 医行為類似 (医師法・医療広告GL)

- 「診断」「判定」(認知症の)
- 「重症度判定」「進行度測定」「進行度判定」
- 「異常を検出」「兆候を発見」(医療的文脈で)
- 「スクリーニング」(医療スクリーニング的文脈で)
- 「○○症状を検知」(医療症状の検知)

> 💡 スコアリングAI機能は **内部品質向上のための仕組み** として実装。家族向け表示・対外訴求では「**会話の様子の記録**」「**会話パターンの変化**」「**ふだんとの違いをお知らせ**」程度の表現を用い、「**進行度**」「**重症度**」「**スコア**」を家族向けUI/コミュニケーションで露出しない。

#### 優良誤認・有利誤認 (景表法)

- 「必ず」「絶対」「100%」「確実」 (効果・サービス品質の断定)
- 「日本一」「業界初」「No.1」 (客観的立証なし)
- 「世界唯一」「他にはない」 (立証なし)
- 「最先端」「最新技術」 (具体的根拠なし)
- 「劇的に」「驚くほど」 (誇大表現)

#### 配慮欠如表現

- 「認知症患者」 → 「**認知症の方**」「**認知症のある方**」
- 「ボケ」「呆ける」「ボケ防止」 → 使用禁止
- 「徘徊」 → 「**ひとり歩き**」(可能なら言い換え)
- 「問題行動」 → 「**行動・心理症状 (BPSD)**」または具体的描写

#### 家族介護者への罪悪感喚起

- 「あなたが見ていないと…」
- 「ご家族の責任で…」
- 「親を一人にしないで…」

> 💡 家族介護者向けマーケティングでは「あなたを責めるトーン」ではなく「あなたを支えるトーン」を選ぶ。
> ただし「**優しく話せず自分を責めているあなたへ**」のような **共感的な現状描写は許容** (LP既存表現)。

## 3. 第2期新機能の表現ルール (重要)

第2期で実装予定の機能は、表現に特に注意が必要。

### 3.1 認知症進行抑制機能

実装事実: 回想法等の学術知見を活用した会話設計・AIによる話題誘導。

| 場面 | NG | OK |
|---|---|---|
| 機能名 | 「認知症進行抑制機能」 | 「**学術的会話設計**」「**脳を使う会話プログラム**」「**非薬物的アプローチを活用したAI会話**」 |
| 効果 | 「進行を抑制」「進行を遅らせる」 | **効果は謳わない**。学術的技法を使っていることのみ訴求 |
| 技法 | 「療法」「治療」 | 「**非薬物的アプローチ**」「**回想法を活用した会話設計**」 |

### 3.2 スコアリングAI・家族レポート機能

実装事実: 会話分析・ライフログ蓄積・話題誘導指示、家族向けナラティブレポート・複数指標逸脱時の通知。

| 場面 | NG | OK |
|---|---|---|
| 機能名 (内部) | (実装はスコアリングで可) | 「**スコアリングAI**」は内部呼称・技術文書のみ |
| 機能名 (対外) | 「進行度スコアリング」「重症度判定」 | 「**会話分析AI**」「**ナラティブレポート機能**」「**変化検知通知**」 |
| 家族向けUI表示 | 「進行度スコア: 72」「危険度: 高」 | 「**会話のようす**」「**ふだんとの違い**」(医療的判定と誤認させない) |
| 緊急通知 | 「異常を検出」「症状悪化」 | 「**ふだんと違う会話の様子をお知らせ**」「**変化があった際の通知**」 |

### 3.3 ライフログ機能

実装事実: 通話メタデータ・会話パターン・生活リズムを蓄積。

| 場面 | 注意点 |
|---|---|
| データ取得の明示 | 「ご本人とご家族の同意のもと会話から間接取得」を必ず併記 |
| 利活用先 | 患者・家族・ケアマネへの提供は事実明示OK。「匿名化ビッグデータの産学連携活用」は **同意取得・匿名化方針を別途明示** |
| 表現 | 「**生活リズムの記録**」「**会話の振り返り**」程度に。「監視」「モニタリング」「監査」等の警察的表現は避ける |

## 4. 必須記載事項

以下を訴求する場合は、必ず根拠/注釈を併記する。

| 訴求 | 必須記載 |
|---|---|
| 「○○名以上のご利用」 | 利用者数の出典・集計時点 |
| 「△△%が満足」 | 調査対象・期間・有効回答数 |
| 「医療法人監修」「松栄会監修」 | 監修内容の具体化 (バリデーション技法・回想法等) |
| 「介護離職1人500万円」等の数値 | 出典 (経産省試算等) |
| 「広島県認知症患者率全国ワースト1位」 | 出典 (厚労省/広島県データ・年次) |
| 「効果がある」系の表現 | 根拠 (調査・データ・出典) |
| お客様の声 | 個人特定情報の同意取得済表記 |
| 比較表現 (他社との比較) | 比較対象の明示・調査時点 |
| キャンペーン情報 | 期間・条件・申込方法 |

## 6. チェック手順

### 6.1 制作時セルフチェック

コンテンツドラフト完成後、以下を確認:

1. **NG表現リスト** にヒットする語が含まれていないか (検索)
2. **慎重な表現** を使っている場合、推奨言い回しに修正したか
3. **必須記載事項** が漏れていないか
4. 「**家族介護者を責めるトーン**」になっていないか
5. 認知症の方を **当事者として尊重した表現** になっているか
6. 第2期機能を扱う場合、**3章のルール** に従っているか

### 6.2 公開前最終チェック

- 法令解釈に不安がある表現は、本規程未収載でも保留にする
- 大規模公開 (プレスリリース・大型広告等) は、社内で複数人レビュー

## 7. 例外承認プロセス

本規程と外れた表現を使いたい場合:

1. なぜその表現を使いたいか・代替案がないかを整理
2. 法的リスク評価 (景表法・薬機法等の観点)
3. 個別判断として承認/却下
4. 承認された場合は本規程の「例外事例」セクションに追記 (将来の参照のため)

## 8. 更新ポリシー

本規程は **固定ではなく、サービス成長・法令変化に応じて更新する**。

### 解禁済 (既に積極訴求OK)

| 領域 | 解禁条件 | 状態 |
|---|---|---|
| 医療法人監修 | 提携契約済 | ✅ 解禁済 (松栄会) |
| ひろしまAIサンドボックス採択 | 事実 | ✅ 解禁済 |

### 将来解禁を検討する領域

| 領域 | 解禁条件 |
|---|---|
| 認知機能への効果訴求 (進行抑制・改善等) | 査読論文・RCT等のエビデンス確立後 |
| 「大学共同研究」訴求 | 共同研究契約締結後・連携先名公表許可後 |
| 「医師監修」訴求 | 医師による監修体制の整備後 |
| 進行度スコアの家族向け可視化 | 医療法務レビュー・適切な責任分界後 |
| 「ライフログのビッグデータ産学連携活用」訴求 | 同意取得スキーム・匿名化方針確立後 |
