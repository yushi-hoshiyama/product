## 1. プレス戦略の全体方針

### 1.1 目的

| 軸 | 目的 |
|---|---|
| **認知** | DELNEというサービス名・「家族の声で応答するAI電話」の概念を業界・地域で広める |
| **信頼** | 医療法人 (松栄会) 監修・ひろしまAIサンドボックス採択など、第三者の裏付けを通じた信頼形成 |
| **送客** | 介護家族 (BtoC) と企業人事 (BtoB) の双方からLP・問い合わせへの流入を生む |
| **採用・連携** | 広島県内の人材採用・追加提携 (自治体・地域包括・企業) の足がかり |

### 1.2 KPI (定量・暫定)

| KPI | 目標 (6ヶ月累計) | 計測方法 |
|---|---|---|
| プレス配信本数 | 6〜8本 | 配信ログ |
| メディア掲載数 (Web + 紙 + 放送) | 累計 30件以上 | クリッピング |
| 主要メディア掲載 (全国紙 / 全国テックメディア / 中国新聞いずれか) | 累計 3件以上 | 同上 |
| LPへのプレス経由流入 | 月平均 200セッション | GA4 (リファラ ※後日設定) |
| 無料トライアル申込のプレス経由比率 | 全申込の 15% 以上 | 申込フォーム流入元 |

> 🔧 GA4 計測タグ・申込フォーム流入元タグは別タスク。SNS企画スレッドのKPI設計と整合させる。

### 1.3 編集方針 (compliance.md / brand-guide.md と整合)

- 「進行抑制」「進行を遅らせる」等の **効果断定は使わない**
- 機能名の対外表現は「**学術的会話設計を活用したAI**」「**非薬物的アプローチ**」を採用
- 「認知症患者」→ 「**認知症のある方**」「**認知症のご家族**」に統一
- 「医療レベル」は使わず、**「医療法人社団 松栄会 監修」** と具体名で訴求
- 数値訴求 (介護離職コスト・認知症患者数等) は **必ず出典を併記**
- トーンは「やさしく、丁寧に、けれど主体的に」(brand-guide.md 2.1)

## 3. 6ヶ月ロードマップ (時期×トピック)

「**月1本の主要リリース + 必要に応じてサブ配信**」を基本ペースに、認知形成 → 信頼形成 → 行動喚起 の順で組む。

| 月 | 主要リリース (本配信) | サブ/連動 | フェーズ |
|---|---|---|---|
| **2026-06** | **T-01 サービス本格提供開始** (+ T-03 採択・第2期スタートを内包) | T-06 松栄会監修を補強情報として | フェーズ① 認知 |
| **2026-07** | **T-06 医療法人社団 松栄会との認知症介護パートナー連携** (単独詳細リリース) | T-02 第1期実証完了レポートを参考資料として併出 | フェーズ② 信頼 |
| **2026-08** | **T-13 企業向けプラン・福利厚生パッケージ正式提供** + **T-14 助成金活用ガイド公開** | T-18 法改正1年振り返り (note 連動記事) | フェーズ③ BtoB訴求 |
| **2026-09** | **T-16 世界アルツハイマー月間連動 — 広島の認知症課題レポート (T-19内包)** | T-09 介サポひろしま協業を地域メディア向けに | フェーズ④ 社会性 |
| **2026-10** | **T-12 プレゼント機能リリース** (個人・親族・企業の3層支払構造) | T-07/T-08 (技術連携) を技術メディア向けに同月配信検討 | フェーズ⑤ 拡張 |
| **2026-11** | **T-17 「介護の日」連動 — 利用者中間報告 (T-05)** | T-10 日本福祉用具供給協会協業 | フェーズ⑥ 行動喚起 |

### 3.1 編集ストーリー (6ヶ月のつながり)

> **6月: 始まりました → 7月: 医療と組みました → 8月: 企業を支えます → 9月: 広島の現実を伝えます → 10月: みんなで支えられます → 11月: もう支えています。**

各月のリリースが次月の文脈を準備する構成。「サービス紹介の連続」にせず、**毎月違う角度**から DELNE を語る。

### 3.2 配信間隔のルール

- **主要配信は原則 月1本** (連続2週以内に複数本出さない)
- **サブ配信は地域メディア・業界メディア限定**、PR TIMES等の全国配信プラットフォームでは出さない
- **同月内に2本主要配信が必要な場合** は、ターゲットメディアを完全に分ける (例: 全国紙 vs テックメディア)

## 4. 主要4本の見出し・リード方向性

ロードマップ中、特に重要度の高い4本について、見出しとリード文の方向性を提示する (本文制作は作業スレッド)。

### 4.1 【2026-06】サービス本格提供開始 (T-01 + T-03)

#### 見出し案 (3本)

1. **「でんわにデルネ」サービス本格提供を開始 — 介護と仕事の両立を支援するAI電話、医療法人社団 松栄会 監修**
2. **家族の声で応答するAI電話「DELNE」、本日サービス開始 — ひろしまAIサンドボックス第2期実証も同時スタート**
3. **広島発、認知症のご家族のお電話に24時間応答するAI「DELNE」が始動 — 介護離職の防止を支援**

> 推奨: **案1** (DELNE を主役にしつつ、松栄会監修を信頼軸として明示)

#### リード方向性

- DELNE が何をするサービスかを **3行以内** で説明 (家族の声 / 24時間 / 介護と仕事の両立)
- 医療法人社団 松栄会 監修・ひろしまAIサンドボックス採択を **第1段落で言及**
- 「介護離職を防ぐ」「ビジネスケアラー307万人」を **数値+出典つき** で配置
- CTA は無料トライアル + 公式LP (https://lp.delne.jp/)

#### 配信先優先度

- 🟢 高: PR TIMES (全業界配信)・中国新聞・NHK広島・RCC・テックメディア (ITmedia/CNET/ASCII)
- 🟡 中: 介護専門紙 (シルバー新報・介護ビジョン)・日経xTECH・福祉系Webメディア

### 4.2 【2026-07】医療法人社団 松栄会との認知症介護パートナー連携 (T-06)

#### 見出し案 (3本)

1. **DELNE、医療法人社団 松栄会と「認知症介護パートナー」として正式連携 — バリデーション技法・回想法を活用したAI会話設計を共同監修**
2. **「家族の声で応答するAI」が医療法人と組む理由 — 松栄会監修のもと、認知症のご家族との会話品質を継続検証**
3. **AIスタートアップ × 医療法人 — DELNEと松栄会が認知症介護領域で共同実証**

> 推奨: **案1** (連携の具体性・技法名で景表法対応)

#### リード方向性

- 「監修」の具体内容を明示 (バリデーション技法・回想法を活用した会話設計の監修)
- **「療法」「治療」は使わず**、「**非薬物的アプローチを活用した会話設計**」と表現
- 松栄会のコメント (代表者談話) を 1段落取る
- 「進行抑制」「進行を遅らせる」等の効果断定は **絶対に書かない**
- 第2期で何を共同検証していくか (会話品質・利用者満足度) を提示

#### 配信先優先度

- 🟢 高: 中国新聞 (広島の医療・地域連携)・医療系専門紙 (m3.com・MedPeer等)・PR TIMES
- 🟡 中: 介護専門紙・地域包括/自治体向け業界誌

### 4.3 【2026-09】世界アルツハイマー月間連動 — 広島の認知症課題レポート (T-16 + T-19)

#### 見出し案 (3本)

1. **【世界アルツハイマー月間】広島県は認知症患者率が高い地域 — DELNEが地域の現実と「電話のもう一つの受け手」の必要性をレポート**
2. **広島県の認知症のある方は約15万人、MCI含めると約28万人 — 家族の電話対応負担をAIで受けとめる取り組み、DELNEが現状を発信**
3. **9月は世界アルツハイマー月間 — 広島発AI電話サービスDELNE、ビジネスケアラー307万人時代のケア構造を考える**

> 推奨: **案1** (社会性 + 当社の役割を両立。ただし「ワースト1位」表現は使わず「高い地域」に和らげる)

#### リード方向性

- **データは必ず出典つき** (厚労省/広島県統計・年次明記)
- 「ワースト」等の挑発的単語は避け、**事実描写** で構成
- 介護家族の電話対応負担 (頻回電話) と、DELNE が果たす役割を **構造的に説明**
- 認知症のある方の **尊厳に配慮** した記述 (「患者」を使わない・「ボケ」「徘徊」使用禁止)
- 自治体・地域包括との連携余地を最後に提示

#### 配信先優先度

- 🟢 高: 中国新聞・NHK広島・広島県政記者クラブ・RCC・全国紙地方版
- 🟡 中: 認知症関連NPO/学会・地方自治体広報

### 4.4 【2026-10】プレゼント機能リリース (T-12)

#### 見出し案 (3本)

1. **「介護を支える側を、もう一人増やす」— DELNE、AI電話サービスをプレゼントできる機能を提供開始**
2. **遠方のご家族・親族・企業から、介護家族へ「DELNE」を贈れる — プレゼント機能リリース**
3. **個人 / 親族 / 企業 / 行政 が支払う4層モデルを完成 — DELNEプレゼント機能で「介護費用の社会化」を一歩前へ**

> 推奨: **案2** (機能ベネフィットがストレートに伝わる。社会論まで広げたい場合は案3)

#### リード方向性

- 「**誰が使うか**」と「**誰が払うか**」を分離した支払構造を 図解レベルで説明
- 遠方在住の介護家族 (ペルソナ③) の **後ろめたさを和らげる** 文脈を中心に
- 「企業導入」「両立支援等助成金」も触れ、BtoB導線も同時に張る
- 罪悪感を煽る表現は使わない (brand-guide.md 9.1)

#### 配信先優先度

- 🟢 高: PR TIMES・テックメディア (ITmedia/TechCrunch Japan)・介護系Webメディア
- 🟡 中: 経済紙 (日経電子版・東洋経済オンライン)・福利厚生関連メディア

## 7. リスク・留意事項

### 7.1 表現リスク

- **「進行抑制」「進行を遅らせる」**: いかなる文脈でも本期間中は **使用禁止**。代替は「学術的会話設計を活用したAI」「非薬物的アプローチ」
- **「医師監修」**: 監修は医療法人 (松栄会) であり「**医療法人社団 松栄会 監修**」と具体名で。医師個人名で監修を訴求する場合は事前合意必須
- **「大学共同研究」**: 締結前は **絶対に書かない** (compliance.md)
- **数値訴求**: 「介護離職9兆円」「広島県認知症患者率」等は **必ず出典・年次** を明記

### 7.2 配信タイミングのリスク

- 大型ニュース (災害・選挙等) と重なった場合、配信延期を判断 (主要メディアに埋もれるため)
- 認知症関連の社会的トピックが重なった場合は逆に **連動報道を狙う** (例: 介護保険改正の議論等)

### 7.3 取材対応

- 取材依頼が来た場合の対応窓口・スポークスパーソンは **別途決定が必要** (本スレッド外・統合企画へ上申)
- 取材時の **NG表現リストカード** (compliance.md ベース) を事前に用意
