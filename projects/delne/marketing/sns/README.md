# SNS

DELNEのSNS投稿をプラットフォーム別に管理する。

## プラットフォーム

| ディレクトリ | 用途 |
| --- | --- |
| `x/` | X (旧Twitter) |
| `instagram/` | Instagram (フィード・リール・ストーリーズ) |
| `facebook/` | Facebook |
| `tiktok/` | TikTok |
| `youtube/` | YouTube (Shorts含む) |

## 運用方針

- 各プラットフォーム内の構造 (日付別 / キャンペーン別 / 投稿フォーマット別) は、運用が回り始めてから判断する
- 投稿テキスト・キャプション・ハッシュタグはMarkdownで管理し、画像/動画は別途ストレージで管理する想定 (大容量バイナリはコミットしない)
