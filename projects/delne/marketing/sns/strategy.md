## 1. ターゲット整理 (3ペルソナとSNS利用傾向)

`projects/delne/product/summary.md` 定義の3ペルソナを SNS 軸で整理する。

| # | ペルソナ | 想定属性 | 主利用SNS | SNSでの行動傾向 | SNSでDELNEに触れる動機 |
|---|---|---|---|---|---|
| ① | 直接介護されている方 (ビジネスケアラー) | 40〜60代、フルタイム就労、子育てと介護の二重負担も | Instagram / X / Facebook | 介護タグ・育児タグ検索、夜間スマホ閲覧多い | 「自分を責めずに済む方法」「同じ境遇の人の経験」 |
| ② | 経営者・人事担当者 | 30〜60代、中小企業含む | Facebook / X / LinkedIn | 業界ニュース・福利厚生・補助金情報を収集 | 「介護離職防止」「両立支援助成金」「導入事例」 |
| ③ | 遠方在住で直接介護が難しい方 | 30〜50代、都市部在住が多い | Instagram / X / LINE | 親の状況を案じてキーワード検索 | 「離れていてもできる支援」「プレゼント形式の支援」 |

地域パートナー (ケアマネ・認知症カフェ・地域包括支援センター) は SNS 主体ではなく **Facebookグループ・地元メディア露出** での接点が中心となる想定。

## 2. プラットフォーム選定

### 採用 / 不採用一覧

| プラットフォーム | 採用判断 | 役割 | 優先度 |

| **note** | ✅ 採用 | 中長文ストーリーテリングと検索流入のストック資産。コラム・実証実験レポート | 🔴 高 |

## 3. プラットフォーム別の目的

### 3.1 X — 一次情報ハブ・記者/介護コミュニティ接点

- **メインKGI**: LP流入 / プレス連動による記者接点獲得
- **ターゲット**: ①ビジネスケアラー / ②人事担当 / メディア関係者 / 認知症コミュニティ
- **コンテンツ軸**:
  - 自社プレスリリース・メディア掲載告知 (中国新聞・RCC・NHK等)
  - ビジネスケアラー向けTips・統計・コラム引用
  - 広島ローカル動向 (認知症施策・自治体連携) のキュレーション
  - 提携先 (松栄会・ひろしまAIサンドボックス) との発信連動
- **頻度**: 週3〜5本 (うち1本はオリジナル長文/連投スレッド)

### 3.2 note — ストーリーテリングと検索流入のストック

- **メインKGI**: 検索流入 → LP / 無料トライアル登録
- **ターゲット**: ①ビジネスケアラー / ③遠方家族 / ケアマネ
- **コンテンツ軸**:
  - 介護家族の心理に寄り添うコラム (「優しく話せなかった夜に」等の現状描写)
  - 学術的会話設計 (バリデーション技法・回想法を活用した会話設計) の解説
  - 第1期実証実験のふりかえり (検証可能な事実のみ)
  - 広島の認知症データと地域課題の解説
  - 企業向け: 介護離職と両立支援助成金の解説
- **頻度**: 月2〜4本 (1本あたり2,000〜4,000字)

### 3.3 Instagram — 世界観・情緒の入口

- **メインKGI**: ブランド認知 / プロフィール経由のLP遷移
- **ターゲット**: ①ビジネスケアラー (特に女性) / ③遠方家族
- **コンテンツ軸**:
  - フィード: ブランドトーン (穏やかなビジュアル・短いコピー)
  - リール: 「あなたの声で応答」の世界観紹介、サービス利用フロー、サンプル応答の擬似体験
  - ストーリーズ: メディア掲載・イベント告知・運営の日常
- **頻度**: フィード週1 / リール月2 / ストーリーズ週2〜3
- **注意**: 認知症の方の顔写真・実音声等は本人/家族同意が取れたケース以外は使用しない。compliance.md 3.3 (ライフログ) の同意原則を全画像/映像にも準用。

### 3.4 YouTube — デモ・アーカイブ

- **メインKGI**: LP流入 / デモ呼びかけからの体験誘導
- **ターゲット**: 全ペルソナ + ②人事担当向け説明動画
- **コンテンツ軸**:
  - デモダイヤルのサンプル応答 (現行のLPサンプル音声を映像化)
  - サービス利用フロー (3ステップ) の解説
  - 代表・医療パートナー (松栄会) インタビュー
  - 企業向け説明会の録画ダイジェスト
- **頻度**: 初期は月1本。動画制作リソースに依存
- **メモ**: 立ち上げ初期は **YouTubeを優先度2** とし、まずは Instagram Reels に同尺コンテンツを再利用する形で運用

### 3.5 Facebook — BtoB・地域コミュニティ

- **メインKGI**: 企業問い合わせ / 地域包括・カフェ運営者との接点
- **ターゲット**: ②経営者・人事 / 地域パートナー
- **コンテンツ軸**:
  - 介護離職・両立支援助成金の解説
  - 中小企業向け導入のメリット (1アカウント単位購入可能等)
  - 広島県内の地域イベント・認知症カフェ連携情報
- **頻度**: 週1〜2本

## 4. 投稿カテゴリ・頻度・口調

### 4.1 投稿カテゴリ (横断)

| カテゴリ | 主目的 | 主使用プラットフォーム | 配分目安 |
|---|---|---|---|
| A. 課題共感 (家族介護者の現状描写) | エンゲージメント・フォロー獲得 | X / Instagram / note | 30% |
| B. 製品理解 (機能・利用フロー・サンプル) | LP流入・トライアル誘導 | Instagram / YouTube / X | 25% |
| C. 信頼性 (医療法人監修・公的採択・メディア掲載) | 信頼形成 | X / Facebook / note | 20% |
| D. 知見提供 (バリデーション技法・統計・両立支援制度) | SEO/被引用・専門性 | note / X | 15% |
| E. 運営/ローカル (広島イベント・パートナー連携) | 地域接点 | Facebook / X | 10% |

### 4.2 口調・トーン (暫定)

ブランドガイドライン (`brand-guide.md` で別途整備中) の正式版に従う。確定までの暫定方針:

- 一人称: 「DELNE」または「私たち」
- 二人称: 「あなた」(家族介護者向け) / 「貴社」(企業向け)
- 文体: ですます調 / 短文中心 / 専門用語には必ず注釈
- 禁止: 感情を煽る記号過多 (!!!・!?)、断定的効果訴求、絵文字の多用 (装飾目的の絵文字は1投稿1個以内)
- 必須: 認知症の方を「**認知症の方**」「**認知症のある方**」と呼ぶ。家族介護者を責めるトーンにしない

### 4.3 表現コンプライアンスの自己チェック

全SNS投稿は `compliance.md` の以下を必ずチェック:

1. NG表現 (「進行抑制」「予防」「治療」等) 不使用
2. 慎重な表現 (「孤独感の軽減」等) は推奨言い回しに置換
3. 必須記載 (出典・期間・条件) の漏れなし
4. 「あなたを責めるトーン」になっていない
5. 第2期機能を扱う場合は compliance.md 3章のルールに従う

### 5.1 主要ハッシュタグ群 (X / Instagram 共通ベース)

| 群 | ハッシュタグ例 | 用途 |
|---|---|---|

- 1投稿あたりのタグ数: X は 1〜3個 / Instagram は 8〜15個

- バズ狙いの無関係タグ (ジャック行為) は禁止

### 5.3 検索流入向けキーワード (note・YouTube)

- 「介護 電話 頻回」「親 電話 何度も」
- 「介護離職 防止 助成金」「両立支援等助成金 中小企業」

- 「広島 認知症 サポート」「ビジネスケアラー 制度」

## 7. KPI設定

### 7.1 立ち上げ初期 (1〜3ヶ月)

**主目的**: 基盤構築と検証可能なベースライン形成。アカウント単独のフォロワー数より、**LP流入と質的シグナル** を重視。

| 指標 | プラットフォーム | 3ヶ月目標 |
|---|---|---|
| フォロワー数 | X | 500 |
| フォロワー数 | Instagram | 300 |
| フォロワー数 | Facebookページ | 100 |
| ビュー数 (累計) | note | 5,000 PV |
| 視聴回数 (累計) | YouTube | 1,000 views |
| LP流入数/月 (SNS全体, GA計測) | — | 月300セッション |
| 無料トライアル登録 (SNS経由, UTM計測) | — | 累計20件 |
| 企業からの問い合わせ (SNS経由) | — | 累計3件 |

### 7.2 中期 (4〜6ヶ月)

| 指標 | 目標 |
|---|---|
| LP流入数/月 (SNS経由) | 月700セッション |
| 無料トライアル登録 (SNS経由, 累計) | 60件 |
| メディア接点 (記者DM・取材依頼) | 累計5件 |
| 企業問い合わせ (SNS経由, 累計) | 10件 |

### 7.3 計測ルール

- 全SNSリンクは UTM パラメータ (`utm_source` / `utm_medium=sns` / `utm_campaign`) を必ず付与
- 投稿管理用スプレッドシート (別途準備) で **テーマ・公開日時・URL・直近の数値** を週次で記録
- 月次で本企画書のKPIに照らしふりかえり (統合企画スレッドへ報告)

### 7.4 「数値以外」の質的シグナル

数値だけで判断しない:

- 介護当事者からの肯定的コメント (compliance.md 観点で問題ない範囲で引用検討)
- ケアマネ・医療関係者からの反応
- メディア関係者からのDM・取材依頼
- 他事業者からの提携相談

## 8. 運用体制・ツール

### 8.1 運用体制 (暫定・少数精鋭運用)

- **コンテンツ責任**: マーケ担当 (本スレッドオーナー)
- **コンプラ確認**: 公開前に compliance.md 6.1 セルフチェック必須。大規模公開 (プレス連動・広告連動) は社内複数人レビュー
- **個別投稿制作**: 投稿1本ごとに作業スレッドを切り、本書の方針を参照
- **将来**: Phase 2 全国展開フェーズで運用代行 or 採用拡大を再検討

### 8.2 ツール (候補・初期は最小構成)

| 用途 | ツール候補 | 初期判断 |
|---|---|---|
| 投稿予約・横展開 | Buffer / Hootsuite / 各SNS純正 | 初期は各SNS純正で運用、本数が増えたら検討 |
| 計測 | Google Analytics (LP流入)・各SNSインサイト | 必須 |
| ハッシュタグ調査 | 各SNSの検索機能、Yahoo!リアルタイム検索 | 必須 |
| デザイン | Figma / Canva | Canva で軽量運用 |
| 動画編集 | CapCut / Premiere Rush | 初期は CapCut |

### 8.3 公開前チェックリスト (毎回)

1. compliance.md NG表現リスト不使用
2. 必須記載 (出典・期間・条件) の漏れなし
3. UTMパラメータ付与
4. ハッシュタグ精査 (配慮欠如タグなし)
5. 画像/動画に第三者が映る場合は同意確認
6. 投稿時間帯 (ターゲット閲覧時間: 朝7〜9時 / 昼12〜13時 / 夜21〜23時) を意識

## 9. 開ける論点 (統合企画への上申事項)

本書を確定するにあたり、統合企画スレッドの判断を仰ぎたい点:

1. **アカウント開設タイミング**: 6月開始でよいか / プレス公開と整合させるか (プレス企画スレッドとの調整)
