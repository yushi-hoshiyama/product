# DELNE Marketing

DELNEのマーケティング活動に関する成果物・ナレッジを管理する。

## カテゴリ構成

上流(方針)から下流(実行)の流れで並ぶ。

| ディレクトリ | 内容 |
| --- | --- |
| `strategy/` | マーケティング戦略 (ポジショニング、KGI/KPI、ロードマップ等) |
| `research/` | マーケティングリサーチ (ペルソナ、競合、市場、インタビュー結果等) |
| `lp/` | ランディングページ (コード・コピー原稿) |
| `ads/` | 広告 (コピー、クリエイティブ、運用メモ) |
| `press/` | プレスリリース原稿 |
| `sns/` | SNS投稿 (プラットフォーム別) |

## 運用方針

- 各カテゴリの構造・テンプレートは、実際にコンテンツを作る過程で ClaudeCode と壁打ちしながら育てる
- 認知症・医療領域の発信は薬機法・医療広告ガイドライン・個人情報の観点で慎重に表現を選ぶ
- 成果物のステータス(ドラフト/公開済/アーカイブ)は必要になった時点で運用ルールを決める
