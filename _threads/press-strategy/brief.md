# プレスリリース企画 — Brief

## 目的

DELNEのプレスリリース全体ロードマップを作成。
**いつ・何を・どんな切り口で** 発信するかを企画する。

## スコープ

- **プレストピック洗い出し**: 出せる素材の棚卸し
  - サービス開始 / 第2期実証 / ひろしまAIサンドボックス採択 / メディア掲載
  - 提携 (松栄会・NTTドコモビジネスX・株式会社AIデータ・介サポひろしま・日本福祉用具供給協会)
  - 料金体系・プレゼント機能 など
- **タイミング計画**: どのトピックをいつ出すか (3〜6ヶ月先まで)
- **配信先設計**: メディア種別 (全国紙 / 地方紙 / 業界紙 / WEBメディア / テックメディア)
- **見出し・リード案**: 主要トピックの見出し・リード文の方向性

## スコープ外

- 個別プレスリリースの **本文制作** (作業スレッドで都度)
- 記者発表会の運営計画
- 記者対応・問い合わせ対応

## 必読ファイル

- `CLAUDE.md`
- `projects/delne/product/summary.md` (実績・パートナー・第2期計画・地域戦略)
- `projects/delne/marketing/strategy/compliance.md`
- `projects/delne/marketing/strategy/brand-guide.md` (**作成され次第**)
- `_threads/README.md`
- `_threads/active.md`
- `_threads/press-strategy/log.md`

## 成果物の格納場所

- ロードマップ: `projects/delne/marketing/press/roadmap.md`
- 個別リリース: `projects/delne/marketing/press/<date>-<topic>.md` (別途作業スレッド)

## 成果物の章立て (推奨・ロードマップ)

1. プレス戦略の全体方針 (目的・KPI)
2. トピック棚卸し (出せる素材一覧)
3. 6ヶ月ロードマップ (時期×トピックマトリクス)
4. 主要3〜5本の見出し・リード方向性
5. 配信先メディアリスト (種別・優先度)
6. 配信オペレーション (PR TIMES / 個別配信 / 記者クラブ等の使い分け)

## 完了定義 (本スレッドとしての一区切り)

- 6ヶ月分のプレスロードマップ案を作成
- 主要3〜5本のトピックについて見出し・リード方向性を提示
- 統合企画スレッドの承認を得る

## 他スレッドへの依頼が想定される事項

- **統合企画**: ロードマップ承認、配信先優先度、予算判断
- **ブランドガイド**: 言葉遣い方針が確定したら参照
- **SNS企画**: プレス公開タイミングとのSNS連動 (将来)
