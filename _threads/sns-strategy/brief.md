# SNS運用企画 — Brief

## 目的

DELNEのSNS運用方針を確立。**プラットフォーム選定・投稿方針・初期コンテンツプラン** を策定する。

## スコープ

- **プラットフォーム選定**: X (Twitter) / Instagram / Facebook / note / YouTube / TikTok 等のうち、
  DELNEのターゲット (ビジネスケアラー・企業人事・地域包括/ケアマネ) にどれが適切か
- **投稿方針**: トピックカテゴリ・頻度・口調・ハッシュタグ運用
- **初期コンテンツプラン**: 最初の1〜3ヶ月の投稿テーマ案
- **KPI設定**: フォロワー数 / エンゲージメント / LP流入 などの指標

## スコープ外

- 個別の投稿文制作 (作業スレッドで都度・本スレッドの担当外)
- ペイド広告 (広告クリエイティブスレッドで別途)
- SNS外のコンテンツマーケ (Owned mediaなど)

## 既定モード

**壁打ちモード** で進める ([`_threads/consulting-mode.md`](../consulting-mode.md) 参照)。
ユーザー = シニアコンサルタント / 意思決定者、Claude = アナリスト兼スパーリングパートナー。
決め打ちで答えない、質問先行、批判的検証、決定事項は明示。

## 成果物の章立て (推奨)

1. ターゲット整理 (3ペルソナとSNS利用傾向)
2. プラットフォーム選定 (採用 / 不採用と理由)
3. プラットフォーム別の目的 (採用したもの個別)
4. 投稿カテゴリ・頻度・口調
5. ハッシュタグ・キーワード戦略
6. 初期3ヶ月のコンテンツプラン (テーマカレンダー)
7. KPI設定
8. 運用体制・ツール

## 完了定義 (本スレッドとしての一区切り)

- プラットフォーム選定の論拠と推奨を提示
- 初期コンテンツプラン (3ヶ月分のテーマ・頻度) を作成
- 統合企画スレッドの承認を得る

## 他スレッドへの依頼が想定される事項

- **統合企画**: プラットフォーム選定の承認・予算判断
- **ブランドガイド**: トーン&マナー方針が確定したら参照
- **プレス企画**: プレス公開時のSNS連動方針 (将来)
